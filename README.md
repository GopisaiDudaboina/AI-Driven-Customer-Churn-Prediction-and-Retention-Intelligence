# 📊 Customer Prediction using Machine Learning

## 🚀 Overview

This project focuses on predicting customer behavior using machine learning techniques across multiple domains such as telecom, banking, and e-commerce. The goal is to analyze data and build models that can make accurate predictions to support business decisions.

## 🧠 Models Used

* CatBoost
* Random Forest
* LightGBM

## 📁 Datasets

* Telecom dataset
* Banking dataset
* E-commerce dataset

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* CatBoost
* LightGBM
* Jupyter Notebook

## 🔍 Features

* Data preprocessing and cleaning
* Feature selection and engineering
* Model training and evaluation
* Comparison of multiple ML models

## 📈 Results

The models were evaluated using standard metrics such as accuracy and performance comparison was done to select the best model for prediction.

## ▶️ How to Run

1. Clone the repository
2. Install required libraries
3. Open the Jupyter Notebook
4. Run all cells

## 📌 Conclusion

This project demonstrates how different machine learning models can be applied to real-world datasets to predict customer behavior effectively.
